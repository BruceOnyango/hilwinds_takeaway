## 1.clone the repo
```bash
git clone https://github.com/BruceOnyango/hilwinds_takeaway
cd hilwinds_takeaway    
```

## 2.create a virtual environemnt (windows) thereafter activate it
```bash
python -m venv venv
venv\Scripts\activate
```

## 2.1.create a virtual environemnt (linux) thereafter activate it
```bash
python3 -m venv venv
source venv/bin/activate
```

## 3. Install dependancies
```bash
pip install -r requirements.txt
```

## 4. Run the pipeline (single command to run everything)
```bash
python main.py
```
